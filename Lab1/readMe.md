# Лабораторная работа №1

Создаем `quest1.py`, `quest2.py`, `quest3.py`

```
touch quest1.py
touch quest2.py
touch quest3.py
```

Делаем файл исполняемыми

```
chmod -x quest1.py
chmod -x quest2.py
chmod -x quest3.py
```

Запускаем

```
./quest1.py |./quest2.py 2> errors.txt |./quest3.py 2> errors.txt
```
