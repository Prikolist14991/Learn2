# Лабораторная работа №2

Создаем `NameWelcome.py`, `name.txt`, `error.txt`

```
touch NameWelcome.py
touch name.txt
touch error.txt
```
Редактируем `NameWelcome.py`

```
nano NameWelcome.py
```

Делаем исполняемым файл `NameWelcome.py`

```
chmod -x NameWelcome.py
```

Запускаем

```
./NameWelcome.py < name.txt
```
